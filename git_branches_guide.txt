Guide des Branches Git 

1. Créer une nouvelle branche : 
  - $ git checkout -b nom_de_la_branche 
  - $ git push origin nom_de_la_branche

2. Changer de branche : 
  - $ git checkout nom_de_la_branche 

3. Voir la liste des branches :
  - $ git branch 

4. Fusionner une branche dans la branche actuelle : 
  - $ git merge nom_de_la_branche_a_fusionner 

5. Supprimer une branche (après fusion) : 
  - $ git branch -d nom_de_la_branche_a_supprimer
  - $ git push origin --delete nom_de_la_branche_a_supprimer

6. Set up une branche avant le premier push :
  - $ git push --set-upstream origin nico
